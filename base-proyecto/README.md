Cliente Vite + React

Comandos:

- Instalar dependencias: `npm install`
- Ejecutar en desarrollo: `npm run dev` (http://localhost:3000)
- Construir: `npm run build`

Nota: Vite está configurado para proxear `/api` al servidor en `http://localhost:4000`.

Header:

- El header usa fuentes `Inter` y `Roboto` (incluidas en `index.html`).
- Añade un logo en `client/public/logo-placeholder.png` o actualiza la ruta en `Header.jsx`.
- El header tiene fondo `#00135A`, altura `60px`, búsqueda centrada, botón CTA y área usuario.
